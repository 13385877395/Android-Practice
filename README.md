Notepad is a simple, bare-bones, no-frills note taking app. Simplify and organize your life!

Features:
• Quickly create and save plain-text notes
• Beautiful, easy-to-use UI - optimized for KitKat
• Share notes to and receive text from other apps
• Auto-saves drafts
• View Mode for notes with clickable links
• Sort notes by date or by name
• Keyboard shortcuts for common actions
• Zero permissions and absolutely zero ads
• Open-source

This application was created to help me learn Android development! Suggestions are always welcome.
